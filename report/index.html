Introduction

  • The website I have developed serves as a hub for local cybersecurity professionals to collaborate, share insights, and stay updated on the latest industry trends. The essential story being told is one of knowledge exchange, skill enhancement, and community-building. The chosen structure is centered around user researches, opinions, and how-to guides.

Accessibility

  • Color Contrast:

    Text has sufficient contrast against the background to ensure readability.

  • Text Resizing:

    Website should allow text to be resized without causing layout issues or content overlaps.

  • Semantic HTML:

    Proper HTML tags used to structure content logically. Headings, lists, and other elements convey the document's structure accurately.

Usability

  • Responsive Design:

    The website incorporates a responsive design approach, ensuring optimal user experience across various devices and screen sizes. Special attention is given to mobile devices. I wrote a script that dynamically adjusts image widths to fit smaller screens. This script intelligently resizes images and selects appropriate fall-back images for devices with limited display capabilities.

  • Mobile-Friendly Navigation:

    The navigation menu is thoughtfully designed for mobile users, employing a collapsible menu structure that allows easy access to various sections without cluttering the screen.

  • Consistent User Interface:

    The user interface maintains consistency across different devices, making it intuitive for users to transition from desktop to mobile without a learning curve.

Learning

  • Cybersecurity Fundamentals:

    As a non-cybersecurity professional, I needed to learn the foundational concepts of cybersecurity to design a meaningful platform. Online courses and industry resources provided valuable insights.

  • Community Management:

    I learning a fair bit about effective online community management strategies, including moderation and conflict resolution techniques. I think this knowladge is essential to create a safe and productive environment for users.

  • ARIA Roles and Attributes:

    I learned about ARIA (Accessible Rich Internet Applications) roles and attributes, which allow me to enhance the accessibility of interactive elements, such as dropdown menus and modal dialogs, for users who rely on screen readers.
